# 8051 LED Blinking Using 13-Bit Timer

## Description
This project demonstrates LED blinking using Timer0 in Mode 0 (13-bit timer) on the AT89C51 microcontroller. The timer generates a delay to blink 8 LEDs continuously. The program is developed in Embedded C using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- 8 LEDs

## Software Used
- Keil uVision
- Proteus 8

## Features
- Uses Timer0 in Mode 0 (13-bit timer)
- Generates delay using the timer
- Blinks 8 LEDs continuously
- Demonstrates timer programming in the 8051 microcontroller

## Files
- 13-bit timer.c – Embedded C source code
- 13-bit timer.uvproj – Keil project file
- STARTUP.A51 – Startup file
